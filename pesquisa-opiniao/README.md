# 📋 Pesquisa de Opinião

## 🎯 Objetivo
O sistema criado tem como objetivo coletar dados para uma pesquisa de avaliação de atendimento ao cliente e exibir os resultados após a conclusão de todas as pesquisas.  

## 👩‍💻 Como utilizar o sistema? 
- Primeiro, insira o nome do participante;
- Segundo, insira a idade do participante;
- Terceiro, insira o número correspondente à avaliação do participante (1 para **EXCELENTE**, 2 para **BOM** e 3 para **RUIM**);
- Repita o processo até que todas as entrevistas previstas sejam realizadas (50) e o sistema exibirá o resultado da pesquisa;

## ⚙️ Como o sistema funciona? 
O sistema coletará os dados inseridos e os armazenará conforme as possibilidades de avaliação citadas acima. Ao fim das entrevistas, ele exibirá o número de respostas **EXCELENTES** e respostas **RUINS** que foram coletadas (como adicional, o sistema também apresentará os resultados em porcentagem).
